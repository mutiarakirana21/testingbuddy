# Testing Tools - Developer Utilities

A comprehensive web-based toolkit for testers and developers featuring 9 essential utilities with modern UI and efficient workflows.

## 🚀 Features

### 1. **Curl Generator**
- Generate one-line curl commands
- Custom headers management
- HTTP methods (GET, POST, PUT, DELETE, PATCH)
- Resolve options
- Flags: `-v`, `-i`, `-k`

### 2. **JSON Formatter**
- Side-by-side input/output layout
- Prettify with 2 or 4 space indentation
- Minify to compress JSON
- **Detailed error detection** with line numbers and syntax suggestions

### 3. **XML Formatter**
- Side-by-side input/output layout
- Prettify and minify options
- **Detailed error detection** with line numbers

### 4. **JSON ↔ XML Converter**
- Bidirectional conversion
- Handles complex structures

### 5. **Text Compare**
- **Sublime-style side-by-side comparison**
- Character-level diff highlighting
- Color-coded changes (green/red/yellow)
- Line numbers on both sides

### 6. **Base64 Encoder/Decoder**
- Instant encoding and decoding
- Real-time conversion

### 7. **Unix Timestamp Converter**
- Live current timestamp
- Bidirectional conversion (timestamp ↔ date)
- Timezone support
- Multiple output formats

### 8. **OCP Environment Variable Search**
- Simplified one-line command generation
- Search by variable name, value, or both
- No cluster URL required
- Namespace and deployment filtering

### 9. **History Management**
- Automatic saving to localStorage
- Search functionality
- Last 100 entries stored

## 📦 Installation & Usage

### Local Development

1. Clone or download this repository
2. Open `index.html` in your browser
3. No build process required!

### Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=YOUR_REPO_URL)

Or manually:

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```
